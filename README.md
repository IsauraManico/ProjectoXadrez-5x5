# ProjetoFinal_IA-I
Jogo Xadrez 5x5


Jogo Xadrez, criado totalmente em Java2D com uma comunicação com Inteligência artificial, de forma a criar um agente automático
capaz  de jogar contra o humano em um tabuleiro reduzido 5x5
